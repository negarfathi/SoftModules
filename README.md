# SoftModules

SoftModules is a software modularization and architecture recovery project that analyzes class-level dependencies, groups related components into higher-level modules, and visualizes the resulting structure through automated package and module diagrams. Built around a C# Windows Forms tool and supported by artifacts such as dependency matrices, DOT cluster files, Bunch outputs, an Understand database, and Rational Rose automation, it is designed to help recover, inspect, and document the architecture of existing software systems.
